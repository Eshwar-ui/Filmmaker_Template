# Independent Documentary Filmmaker HTML Template

Production-ready, responsive HTML/CSS/JavaScript template for documentary filmmakers and creative studios.

## Template Highlights
- Two homepages (`Home 1` and `Home 2`) with navbar dropdown
- Core pages aligned with template guide requirements
- Dark/light mode toggle
- RTL stylesheet support
- Accessible navigation and form validation
- Reusable UI components (buttons, cards, forms, badges)

## Quick Start
1. Open `pages/index.html` in your browser.
2. Replace placeholder content and links.
3. Update SEO meta tags, canonical URLs, and social URLs.

## Customization Notes
- Global design tokens are in `assets/css/style.css` (`:root` variables).
- Replace demo media with optimized WebP assets for production.
- Use `documentation/customization.md` for detailed content replacement points.
