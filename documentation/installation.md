# Installation Guide

## Requirements
- Any modern browser (Chrome, Firefox, Safari, Edge)
- Any code editor
- No build tools required

## Setup
1. Extract the template folder.
2. Open `pages/index.html` in your browser.
3. Start editing HTML/CSS/JS directly.

## Important Folders
- `pages/` - all page templates
- `assets/css/` - `style.css`, `dark-mode.css`, `rtl.css`
- `assets/js/` - `main.js`
- `documentation/` - setup, customization, credits, changelog, support

## Core Pages
1. `pages/index.html` - Home 1
2. `pages/home-2.html` - Home 2
3. `pages/about.html` - About
4. `pages/services.html` - Services
5. `pages/contact.html` - Contact
6. `pages/404.html` - 404
7. `pages/coming-soon.html` - Coming Soon

## Optional Portfolio Page
- `pages/films.html`

## Deploy
Upload the full folder to Netlify, Vercel, GitHub Pages, or any static host.
