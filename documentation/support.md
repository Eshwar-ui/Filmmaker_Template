# Support

## Getting Help
- Email: `contact@marcusholloway.com`
- Response time target: within 48 hours on business days.

## Before Contacting Support
- Confirm file paths are unchanged.
- Clear browser cache after CSS/JS updates.
- Validate custom HTML with W3C validator.
- Check browser console for integration errors after adding third-party forms/APIs.

## Scope
Support includes template setup guidance, structure questions, and customization direction.
Support does not include custom feature development or server-side integration builds.
